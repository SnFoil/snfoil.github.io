HPR Flight Computer

A PCB designed to allow various Adafruit breakout boards to communicate with a Raspberry Pi Pico during HPR model rocket launches.

Summary / Table of Contents

Throughout this project, I learned more about interfaces such as I2C and SPI. I also improved my PCB layout skills and EDA software familiarity.

Table of Contents

  • Initial Circuit Design
  • PCB Layout and Design
  • Choosing Parts and Ordering the PCBs
  • Final Thoughts

Initial Circuit Design

When first designing this circuit, I first came up with a list of things I wanted to accomplish with it. I knew I wanted one or more sensors to collect data during an HPR (high powered rocketry) model rocket launch, and so I got started finding parts.

I found all of the sensors that are used in this project in the form of breakout boards from Adafruit, which are basically pre-assembled PCBs with the sensors and supporting circuitry on them with easy-to-access pins.

PCB Layout and Design

For the PCB design and layout, I used KiCAD. I've used this program before, so that's why I picked it over other programs like Autodesk Eagle.

The schematic is simple. Various pins from each breakout board were connected with eachother, along with some LED circuitry. I decided to go with some small 0603 components just because I was curious how difficult it would be to solder them by hand (hint: it's pretty tough).

Choosing Parts and Ordering the PCBs

All of the individual components I purchased were done so through Digikey. If you're interested in the exact parts used, there's a list in the Github repo for this project (click me!).

I ordered my PCBs through JLCPCB. They had a very quick turn around and my PCBs came back just as I designed them.

Final Thoughts

I learned a lot doing this project. I brushed up on my Python skills, used 3 I2C devices on a single bus, and actually put together a final product that I'm proud of.

Although I would consider this project a success, there is definitely room for improvement and things that I want to change. A version 2 is already in development.
